# 회전 시간 연장 업데이트
## 📅 2024-11-12

### 🎯 업데이트 목적
라인 트레이서가 좌회전/우회전 시 회전 각도가 부족하여 라인을 제대로 찾지 못하는 문제를 해결하기 위해 회전 지속 시간을 늘림.

### ✅ 변경 내용

#### 1. **수동 제어 (키보드 입력)**
- **위치**: 교차로 모드 및 라인 이탈 시
- **이전**: 1.0초
- **변경**: 1.2초
- **영향 받은 라인**:
  - 교차로 모드: 687번, 697번 라인
  - 라인 이탈 시: 756번, 763번 라인

```python
# 변경된 코드
motor_forward()
time.sleep(0.5)  # 직진으로 접근
motor_left(1.0)  # 또는 motor_right(1.0)
time.sleep(1.2)  # 회전 시간 (기존보다 길게)
```

#### 2. **자동 제어 (객체 인식)**
- **위치**: 표지판 인식 시 자동 회전
- **이전**: 1.0초
- **변경**: 1.5초
- **영향 받은 라인**: 329번, 345번 라인

```python
# 변경된 코드
motor_forward()
time.sleep(0.5)     # 코너 접근
motor_left()        # 또는 motor_right()
time.sleep(1.5)     # 회전 시간 늘림 (1.0 → 1.5)
motor_forward()
time.sleep(0.5)     # 라인 복귀
```

### 📊 회전 시간 비교표

| 제어 방식 | 시나리오 | 이전 시간 | 변경된 시간 | 증가율 |
|----------|----------|----------|------------|--------|
| 키보드 | 교차로 모드 | 1.0초 | 1.2초 | +20% |
| 키보드 | 라인 이탈 | 1.0초 | 1.2초 | +20% |
| 객체 인식 | 표지판 감지 | 1.0초 | 1.5초 | +50% |

### 🔧 추가 조정 가이드

회전이 여전히 부족한 경우:
```python
# 수동 제어용 (현재 1.2초)
time.sleep(1.4)  # 더 긴 회전

# 자동 제어용 (현재 1.5초)
time.sleep(1.8)  # 더 긴 회전
```

회전이 너무 과도한 경우:
```python
# 수동 제어용 (현재 1.2초)
time.sleep(1.1)  # 약간 짧게

# 자동 제어용 (현재 1.5초)
time.sleep(1.3)  # 약간 짧게
```

### 💡 향후 개선 제안

1. **동적 회전 시간**: 트랙 특성에 따라 자동 조절
2. **각속도 센서**: IMU/자이로스코프 활용한 정확한 회전 각도 제어
3. **속도별 조정**: 주행 속도에 따른 회전 시간 자동 조절
4. **학습 기반**: 성공적인 회전 데이터 수집 후 최적값 학습

### ⚠️ 주의사항

- 회전 시간을 너무 길게 설정하면 과도한 회전으로 반대편 라인을 놓칠 수 있음
- 모터 성능과 배터리 상태에 따라 최적 시간이 달라질 수 있음
- 트랙의 곡률과 폭에 따라 조정 필요
